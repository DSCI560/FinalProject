# Milestone 1 Demo Script

## Goal

Record a short technical milestone video that shows a complete end-to-end workflow for approximately 25-30% of the final AI Cohort Assistant system.

## Recommended video length

4 to 6 minutes.

## Demo narration script

### 1. Opening

"Hello, we are presenting Technical Milestone 1 for our SCI-560 final project, **AI Cohort Assistant**. Our project is a domain-specific AI-enhanced group chat system designed to help cohort members collaborate, share resources, and get grounded AI support using uploaded documents and embeddings."

### 2. Problem statement

"The main problem we are solving is that cohorts and project teams often rely on scattered chat messages, disconnected files, and repeated questions. Our system brings communication and AI-guided support into one shared platform."

### 3. Milestone scope

"Since this is Milestone 1, our goal is not to show the full final product yet. Instead, we are demonstrating the MVP foundation, which covers roughly 20 to 40 percent of the planned system, specifically the core web chat workflow and AI retrieval pipeline."

### 4. Show login or entry flow

"Here, a user enters the system and joins the cohort chat environment."

### 5. Show group chat

"This is the shared group chat interface where users can communicate with each other in real time."

### 6. Show file upload

"Users can upload cohort-specific documents such as schedules, guides, notes, or instructions. These files become part of the knowledge base used by the AI assistant."

### 7. Explain embeddings briefly

"After upload, the system processes the document, breaks it into chunks, and stores embeddings so the AI can retrieve relevant information instead of answering from generic memory alone."

### 8. Ask a grounded question

"Now we ask a question that depends on the uploaded document, such as a schedule question, requirement clarification, or cohort policy question."

### 9. Show AI answer

"The assistant retrieves the most relevant content from the embedded knowledge base and responds directly inside the group chat. This demonstrates the core value of our system: domain-specific AI support within a collaborative chat environment."

### 10. Close with roadmap

"For later milestones, we plan to expand this into a more complete platform with stronger UX, mobile/app support, richer file handling, image-related features, business model validation, and production-level polish. Milestone 1 establishes the technical foundation for that roadmap."

## What should be visible on screen

1. App entry or login page.
2. Cohort or group chat room.
3. Example user messages.
4. File upload action.
5. AI question in the chat.
6. AI answer tied to uploaded material.
7. Optional short slide with milestone roadmap.

## Quick checklist before recording

- make sure the demo path works without errors,
- prepare one clean test file for upload,
- prepare one question whose answer is clearly in the uploaded file,
- close unrelated tabs and windows,
- zoom the browser so text is readable in the recording,
- rehearse once to keep the video concise.
