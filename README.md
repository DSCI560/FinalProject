# AI Wedding Planner - Milestone 1 Demo

This is a dependency-free web prototype for your first milestone video.

## What works

- wedding workspace entry flow,
- shared planning chat UI,
- user messaging,
- text file upload,
- image upload and display,
- document chunking,
- local vector-style retrieval for grounded AI answers,
- AI replies inside the chat when you ask with `@ai`.

## How to run

### Easiest option

Open [index.html](C:/Users/Spiderman/Desktop/DSCI%20560%20Final%20project/index.html) in a browser.

### Demo flow

1. Click `Open Wedding Chat`.
2. Click `Load Sample Doc`.
3. Ask:
   - `@ai what time does the ceremony start?`
   - `@ai who is the photographer?`
   - `@ai when does guest arrival begin?`
4. Upload an image to show decor inspiration or picture display support.
5. Send a normal planner or vendor message to show shared chat behavior.

## Important note

This version is a local milestone prototype. It does **not** use a real hosted backend or live LLM API yet. Instead, it demonstrates the product workflow with local retrieval logic so you can record a working Milestone 1 video from this folder.
